Про Lenovo Ideapad 110 - 15ibr

Описание 15.6" Ноутбук Lenovo IdeaPad 110-15IBR черный 15.6"

Ноутбук Lenovo IdeaPad 110-15IBR черного цвета – модель для людей, которым необходимо устройство для работы и стандартных развлечений. Поставляется он с операционной системой Windows 10. Вы оцените удобство ОС и ее стабильность, сможете полноценно пользоваться различными функциями.

В Lenovo IdeaPad 110-15IBR используется привычная клавиатура для данных устройств. Она очень удобная и прекрасно подходит для печати. Диагональ экрана составляет 15.6 дюйма, матрица выполнена по технологии TN+film. Углы обзора хорошие для комфортной работы с ноутбуком, разрешение – HD+.

Оперативная память DDR3L распаяна по материнской плате, общее количество ОЗУ – 4 ГБ. Для хранения основной информации используется жесткий диск с объемом 500 ГБ, чего будет достаточно большинству пользователей. Присутствует карт-ридер для подключения внешних накопителей и копирования данных.

Основой устройства является процессор Celeron N3060 с частотой 1.6 ГГц. Если нагрузка серьезная, то система автоматически повышает этот параметр до 2.56 ГГц на некоторое время. Количество ядер – два, обеспечивается оптимальная производительность. Графический адаптер встроен в процессор.
